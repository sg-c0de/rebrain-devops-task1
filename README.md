# rebrain-devops-task1

> The learning project for mastering **git**

This project contains custom nginx configuration file

## Install

* Install nginx

1. For debian branch

```
$ apt-get install nginx
```

2. For RHEL branch

```
$ yum install nginx
```

* Copy/create symlink to /etc/nginx.conf fromi file in this repo
